# Health Analyser 
A React-based health alert dashboard that shows health trends, alerts, and history in a clean and simple UI.
This project is focused on visualizing health-related data, detecting abnormal values, and helping users understand their health patterns over time.

## Features
- Health alerts based on abnormal readings
- Trend visualization over time
- Health history tracking
- Clean and responsive UI
- Modular React component structure

## Tech Stack
- React (Vite)
- JavaScript
- CSS
- ESLint
